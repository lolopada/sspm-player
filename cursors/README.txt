Place PNG images here to use as a custom cursor instead of the default dot.

Accepted format: .png

Tips:
  - A square image (e.g. 128x128 or 256x256) works best.
  - Use a transparent background (RGBA) so the cursor looks good over any scene.
  - The image is displayed as-is, without any tinting — use a white or colored design.
  - The cursor is centered on the mouse / stylus position.
  - Displayed size: ~15% of the screen height (approx. 80 px at 540p, 160 px at 1080p).

To select a cursor: menu -> Options -> "Cursor"
Choose from "Default" or any file in this folder.

Example: drop "orb.png" here, then pick "orb.png" in Options.
